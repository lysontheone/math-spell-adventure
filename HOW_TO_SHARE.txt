📚 Math & Spell Adventure - Sharing Instructions 📚

Thank you for sharing this educational game with friends and family! Here's how to share it:

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📦 PACKAGING THE GAME
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
1. Locate the "math-spell-adventure" folder on your computer
2. Right-click the folder and select "Send to" > "Compressed (zipped) folder"
3. You'll get a new file named "math-spell-adventure.zip"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📤 SHARING OPTIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Choose the easiest way to share:

📧 EMAIL:
- Attach the ZIP file to an email
- Add a friendly message about the game
- Send to your friends!

☁️ CLOUD STORAGE:
1. Upload the ZIP file to Google Drive, OneDrive, or Dropbox
2. Right-click the file and select "Share"
3. Choose "Anyone with the link can view"
4. Copy and share the link

💾 USB DRIVE:
1. Insert a USB drive
2. Copy the ZIP file to the USB
3. Give the USB to your friend

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎮 INSTRUCTIONS FOR PLAYERS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
When your friends receive the game, they should:

1. Extract the ZIP file:
   - Right-click the ZIP file
   - Select "Extract All..."
   - Choose a location to extract to
   - Click "Extract"

2. Open the extracted folder

3. Double-click "index.html" to start playing!
   - No installation needed
   - Works offline
   - Safe for kids

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
